# Yuval's 12-Week DV/ASIC Study Plan

**Goal:** Go from HDLBits-level SystemVerilog to building portfolio projects with UVM testbenches, a RISC-V CPU, and communication protocols.

**Start date:** ___________
**Target completion:** 12 weeks from start

## Resources (All Free)

| Resource | URL | Used For |
|----------|-----|----------|
| Verification Academy | https://verificationacademy.com | SV + UVM video courses |
| ChipVerify | https://chipverify.com | SV + UVM quick tutorials |
| EDA Playground | https://edaplayground.com | Run UVM in browser with commercial simulators |
| Nandland | https://nandland.com | UART/SPI reference designs |
| RISC-V Spec | https://riscv.org/technical/specifications | ISA reference |
