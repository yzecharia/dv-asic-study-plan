# LinkedIn Headline Patterns

**Category**: Soft skills · **Used in**: W2, W6, W10, W14, W18 · **Type**: authored

Recruiters scan your LinkedIn headline in ~7 seconds. The headline
is what shows up in search results and connection invitations. Bad
headline = invisible. Good headline = invitation to look at the
About section.

## Three patterns to A/B over a month

### Pattern A — role-targeted, technical (recommended for first iteration)

```
Junior Design Verification Engineer | SystemVerilog · UVM · RISC-V |
Building a 20-week portfolio in the open
```

**Why it works**: search-engine-friendly (recruiters search by
"design verification engineer" exactly), shows technical fluency,
shows commitment via "in the open."

### Pattern B — outcome-focused

```
Verifying RTL the way I'd want it verified | UVM · SVA · CDC |
github.com/yzecharia
```

**Why it works**: voice of someone who has read PRs. Calls out
craftsmanship. Risky if About section doesn't deliver — but shines
if it does.

### Pattern C — story-driven (most distinctive, highest variance)

```
From HDLBits to NVIDIA-tier — autodidact studying RTL/DV in public,
1 commit at a time
```

**Why it works**: humans read story; recruiters scrolling through
hundreds of profiles remember it. Risky because some recruiters
filter on stricter keyword matches (Pattern A wins those filters).

## Don't use

- Vague: "Aspiring engineer passionate about hardware."
- Long: "Fourth-year electrical engineering student with experience
  in Verilog, SystemVerilog, UVM, RISC-V, and looking for an
  internship..." — cut the fat.
- Buzzword soup: "Hardware enthusiast | Innovation | Excellence."

## Iteration cadence

Per `docs/POWER_SKILLS.md`:

| Wk | Action |
|---|---|
| W2 | Write Pattern A. Set as live headline. |
| W6 | Try Pattern B for 2 weeks. Note any change in profile views. |
| W10 | Try Pattern C for 2 weeks. Note same. |
| W14 | Pick winner; refine wording. |
| W18 | Final iteration with strongest portfolio repo named. |
| W20 | Lock the headline that's pulling the most recruiter outreach. |

LinkedIn shows you "profile views in the last week" — use that as
the (noisy) feedback signal.

## About section coupling

The headline buys you a click. The About section closes. Three
short paragraphs — current focus, concrete portfolio, target role.
See `docs/POWER_SKILLS.md` §2 for structure.
